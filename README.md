# -SSM-maven-ajax-c3-h5-Js-
这是一款基于Spring、Mybatis、SpringMVC实现的简单火车票售票与管理系统，其中结合了maven、ajax、html5、css3、JavaScript等技术；适合期末课程设计使用！项目非常简单且易于理解！
（1）关于数据库：
  使用的是mysql 5.7.15，其脚本文件以及放置在项目的web-app目录下，可直接生成相应数据库。
